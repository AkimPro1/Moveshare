Place your design images here and name them exactly as below so the site will load them automatically.

Recommended filenames (drop your PNG/JPG files into this folder):

- `hero.jpg`            — Hero background (large, wide image, e.g. 1920x800)
- `hero-illustration.png` — Decorative illustration (optional)
- `car-suv.png`        — Car image (transparent background ok)
- `car-neon.png`       — Secondary car image (transparent background ok)
- `map-pin-1.jpg`      — Map / location image for features

If you prefer different filenames, update `src/pages/Home.tsx` to point to the files you uploaded.

After adding images, restart the dev server if necessary:

```powershell
cd moveshare-frontend
npm run dev
```

If an image is missing, the page will show a subtle placeholder.
