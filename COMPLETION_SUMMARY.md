### Design System Questions

- `DESIGN_SYSTEM.md` - Color, typography, sizing
- `QUICK_REFERENCE.md` - Quick lookup charts
- `DESIGN_MODERNIZATION.md` - Implementation details

### Code Customization
1. **Change Colors**: Modify CSS variables in `index.css`
2. **Adjust Shadows**: Update shadow elevation values
3. **Speed Animations**: Change duration (keep 300-400ms range)
4. **Add Gradients**: Use 135deg angle for consistency

### Troubleshooting
- **Choppy animations**: Check GPU acceleration (transform-only)
- **Colors too bright**: Reduce gradient opacity
- **Shadows not visible**: Check z-index and positioning
- **Text not readable**: Verify color contrast
