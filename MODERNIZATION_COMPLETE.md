## How to Use the New Design System

### 1. Apply Design Tokens
```css
background: linear-gradient(135deg, var(--ms-primary) 0%, var(--ms-accent) 100%);
box-shadow: 0 10px 30px rgba(0, 212, 255, 0.3);
border-radius: var(--ms-radius-md);
transition: all 0.3s cubic-bezier(0.4, 0.0, 0.2, 1);
```

### 2. Component Styling
- Copy gradient patterns from documented components
- Use color variables for consistency
- Apply shadow levels matching elevation
- Include animation timing for smoothness

### 3. Extend the System
- Add new colors to CSS variables (index.css)
- Create component variants using gradients
- Combine shadow levels for custom effects
- Use animation timing functions consistently

## Maintenance & Support

### Regular Tasks
1. Keep design system documentation updated
2. Maintain CSS variable consistency
3. Test new components against design system
4. Monitor animation performance
5. Verify accessibility compliance

### Common Customizations
- **Change Colors**: Update CSS variables in index.css
- **Adjust Shadows**: Modify shadow elevation values
- **Speed Animations**: Change animation duration (but keep 300-400ms range)
- **Add Gradients**: Use 135deg angle for consistency
- **Update Typography**: Modify Sora/Inter imports

### Troubleshooting
- **Animations choppy**: Check GPU acceleration (use transform only)
- **Colors too bright**: Reduce gradient opacity
- **Shadows not visible**: Check z-index and positioning
- **Text not readable**: Verify color contrast (WCAG AA)
- **Mobile issues**: Test touch events and responsive sizes
